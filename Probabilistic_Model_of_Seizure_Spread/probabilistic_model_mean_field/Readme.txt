Code repository 
Paper: Criticality in probabilistic models of spreading dynamics in brain networks: epileptic seizures
Authors: S Amin Moosavi, Wilson Truccolo
Journal: PLoS Computational Biology (2023)

This part of the repository contains the Matlab code for the simulation of mean-field dynamics of the probabilistic model

Run Mean_field_example.m for an example of mean-field dynamics with parameters as specified in the paper.
