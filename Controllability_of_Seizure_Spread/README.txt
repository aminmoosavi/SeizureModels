Code repository 
Paper: Controllability of nonlinear epileptic-seizure spreading dynamics in large-scale subject-specific brain networks
Authors: S Amin Moosavi, Jordan S Feldman, Wilson Truccolo
Journal: Nature Scientific Reports (2025)

This part of the repository contains:

(1) The compressed (zip) datasets for each of the 5 patient-specific connectivity networks required to reproduced the results in this paper. These datasets have been used and publicly shared before in Moosavi, Jirsa, Truccolo (PLoS ONE, 2022). Each dataset contains ascii files specifying 

(a) the name of the brain areas (areas.txt)
(b) non-normalized connectivity weights (weights.txt)
(c) tract lengths of white-matter fibers (tract_lengths.txt)

The additional files centres.txt, average_orientations.txt, cortical.txt, are provided, but were not used in this study.

(2) Code for simulation of the patient-specific epileptor network models

Run /code/main.ipynb for an example of the simulation of an epileptor network model.
